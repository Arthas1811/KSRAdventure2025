# Running instructions
* Unzip the folder
* Open the scene "Construction" in unity and click on play
# External help
* Physics and collider implementation followed the following tutorial: https://www.youtube.com/watch?v=7LVOeNalTFA
* 3D assets were reconstructed from 2D images using this platform: https://aidemos.meta.com/segment-anything/gallery/
* Some parts of the code were reused from KSRAdventure2025, as the same tasks were required (these sections are clearly marked with comments)
* The end-game cutscene and the images for the buttons were generated using AI tools
* Custom high-precision collider geometry was generated via AI written C# scripts. These provide more detail than Unity’s default collider components
* The math used in the "AlignPreviewObjectWithCamera" function was explained with the help of AI since the concepts needed to achieve the intended behavior were not yet known
* The sound effects were taken from this website: https://pixabay.com/
